Belly Button Biodiversity
A Web Dashboard for a SQL Database of Belly Button Microbes

On the back-end, a SQL database about various samples of microbes found in human navels is accessed and made available in JSON form. On the front-end, a selectable (by sample) dashboard is produced that visualizes the different types of microbes in that sample.

The project is deployed on Heroku here.

Tools:

  • Flask (Python)
  • Pandas (Python)
  • SQLAlchemy (Python)
  • D3 (Javascript)
  • Plotly (Javscript)
  • Bootstrap CSS
  • HTML
  • SQLite Database
